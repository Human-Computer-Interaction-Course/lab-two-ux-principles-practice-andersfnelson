Usability Reccomendations

I used Google's PageSpeed Insights and Pingdom on the Brookfield Zoo website.

  1. Reduce the use of third party code: Google says that third party code blocked the main thread for 760 milliseconds. The site could improve performance by loading this code after the main site components have loaded.
  2. Another thing that could be done to improve performance would be reducing the use of URL redirects. It looks like there are many instances of the browser being told to go to another location when the page is loaded. This means that nothing can be rendered until the HTML document arrives.

Accessibility Reccomendations

Using the WAVE accessibility tool, I identified the following accessibility improvements:

  1. Make sure that all images have an alternative text for screen readers. 8 images were identified without the alt text.
  2. The tool also identified four empty links, which can create confusion for people using screen readers. These links should be removed or fixed.
